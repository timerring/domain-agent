# Domain Agent Frontend

React + TypeScript 前端应用

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

## 技术栈

- React 18
- TypeScript
- TailwindCSS
- React Query
- Axios
- Vite

## 项目结构

```
frontend/
├── src/
│   ├── components/      # UI 组件
│   ├── services/        # API 服务
│   ├── App.tsx          # 主应用
│   └── main.tsx         # 入口文件
└── package.json
```
